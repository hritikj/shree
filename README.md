# Shree Project

Shree is a PHP-based web application with a MySQL backend. This project demonstrates a basic e-commerce or product management system.

## Features

- User registration and login
- Product listing and management
- Shopping cart functionality
- Order updates
- Integration with MySQL database using `shree.sql`
- Frontend using HTML, CSS, and JavaScript


## Setup Instructions

1. **Clone the repository**
2. **Install Apache, PHP and MySQL**
3. **Run the Database setup script**
4. **Start Apache and MySQL**
5. **Access the Project**
